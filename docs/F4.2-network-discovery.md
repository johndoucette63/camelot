## Overview

Implement automated LAN discovery (nmap-based) and a persistent device inventory. The advisor should know every device on 192.168.10.0/24, what it is, and when it was last seen. Users can annotate devices with roles and descriptions.

## User Stories

### US-1: Scan LAN and discover devices
**As a** Camelot admin
**I want** the advisor to scan 192.168.10.0/24 and find all connected devices
**So that** I have a complete picture of what's on my network

#### Acceptance Criteria
- Scheduled nmap scan discovers devices (hostname, IP, MAC, vendor)
- Scan runs on a configurable interval (default: every 15 minutes)
- New devices detected since last scan are flagged
- Scan results persisted to PostgreSQL

### US-2: View device inventory in the dashboard
**As a** Camelot admin
**I want** a device list page showing all discovered devices
**So that** I can see at a glance what's connected

#### Acceptance Criteria
- Dashboard page lists all devices with IP, hostname, MAC, vendor, last seen
- Online/offline status indicated (green/gray dot)
- Known Camelot devices (HOLYGRAIL, Torrentbox, NAS, etc.) highlighted
- List is sortable and filterable

### US-3: Annotate devices with roles and tags
**As a** Camelot admin
**I want** to assign a role, description, and tags to any device
**So that** the advisor knows what each device does beyond its hostname

#### Acceptance Criteria
- Click a device to edit: role (server, IoT, workstation, etc.), description, tags
- Annotations persisted to database
- Pre-populated annotations for known Camelot devices
- Annotations visible in device list and available to AI chat context

### US-4: Detect new and missing devices
**As a** Camelot admin
**I want** to be alerted when a new device appears or a known device goes offline
**So that** I notice unauthorized devices or outages quickly

#### Acceptance Criteria
- New device events logged with timestamp
- Known devices missing for >2 scan cycles flagged as offline
- Event history viewable in dashboard
- Events available as context for AI chat and recommendations
