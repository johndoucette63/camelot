## Overview

Improve torrent quality by adding paid private indexers and tuning quality profiles in Sonarr and Radarr. This phase is independent of HOLYGRAIL and can start anytime.

## User Stories

### US-1: Evaluate and select paid indexers
**As a** media consumer
**I want** to research private trackers and indexers (IPTorrents, TorrentLeech, etc.)
**So that** I get better quality torrents with more seeders

#### Acceptance Criteria
- At least 3 paid indexers evaluated (cost, content catalog, invite requirements)
- Decision documented with rationale
- Account(s) created on selected indexer(s)

### US-2: Add indexers to Prowlarr
**As a** media consumer
**I want** new indexers configured in Prowlarr with API keys
**So that** Sonarr and Radarr can search them automatically

#### Acceptance Criteria
- Indexers added to Prowlarr with valid API keys
- Sync verified: Sonarr and Radarr pull results from new indexers
- FlareSolverr still handling CloudFlare-protected indexers
- Search results show improved availability

### US-3: Tune Sonarr/Radarr quality profiles
**As a** media consumer
**I want** quality profiles configured to prefer high-quality releases
**So that** downloads default to good quality without manual intervention

#### Acceptance Criteria
- Quality profiles reject: CAM, TS, low-bitrate encodes
- Preferred: Bluray remux/encode, WEB-DL, HDTV (in priority order)
- Size limits set to prevent excessively large downloads
- Upgrade until cutoff configured (e.g., upgrade to Bluray from HDTV)

### US-4: Optimize Deluge connection settings
**As a** media consumer
**I want** Deluge tuned for optimal download performance
**So that** torrents download faster and seeding is well-managed

#### Acceptance Criteria
- Max connections and per-torrent limits tuned for the Pi's resources
- Encryption set to forced for privacy
- Queue management: max active downloads, seeding ratio limits
- Settings documented in INFRASTRUCTURE.md
