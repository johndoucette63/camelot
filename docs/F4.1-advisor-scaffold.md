## Overview

Stand up the foundational application structure for the Network Advisor: a FastAPI backend, React + Tailwind frontend, PostgreSQL database, and Docker Compose to orchestrate them. No business logic yet — just the skeleton that all subsequent F4.x features build on.

## User Stories

### US-1: Scaffold FastAPI backend
**As a** developer
**I want** a FastAPI project with a health endpoint, structured routing, and Docker container
**So that** there's a working backend ready for feature development

#### Acceptance Criteria
- `advisor/backend/` contains a FastAPI app with `main.py`, routers, and models directories
- `/health` endpoint returns `{"status": "ok"}`
- Dockerfile builds and runs the backend
- Requirements file lists pinned dependencies

### US-2: Scaffold React frontend
**As a** developer
**I want** a Vite + React + Tailwind CSS project with a Docker container
**So that** the frontend is ready for dashboard and chat UI development

#### Acceptance Criteria
- `advisor/frontend/` contains a Vite + React + TypeScript project
- Tailwind CSS configured and working
- Dockerfile with multi-stage build (build → nginx serve)
- Dev mode accessible for local iteration

### US-3: Deploy PostgreSQL database
**As a** developer
**I want** a PostgreSQL container with an initial schema
**So that** the backend has persistent storage for devices, services, and alerts

#### Acceptance Criteria
- PostgreSQL container defined in Docker Compose
- Database `advisor` created with `john` user
- Initial migration or schema script runs on first boot
- Data persisted via Docker volume

### US-4: Orchestrate with Docker Compose
**As a** homelab owner
**I want** `docker compose up` to start the entire advisor stack
**So that** deployment is a single command

#### Acceptance Criteria
- `advisor/docker-compose.yml` defines backend, frontend, and postgres services
- Frontend proxies API calls to backend
- All services on a shared Docker network
- Accessible at `http://holygrail:<port>` from the Mac
- Environment variables for database URL, Ollama URL, etc.
