## Overview

Wipe HOLYGRAIL (currently Windows 11 Pro) and install Ubuntu Server 24.04 LTS as a headless basement server. This is the prerequisite for all downstream infrastructure work.

## User Stories

### US-1: Back up HOLYGRAIL Windows installation
**As a** homelab owner
**I want** to back up anything worth keeping from Windows 11 before wiping
**So that** I don't lose important data or configurations during the migration

#### Acceptance Criteria
- Inventory of files/configs worth preserving is documented
- Backup copied to NAS (192.168.10.105) via SMB
- Confirmation that nothing critical remains only on the local disk

### US-2: Create bootable Ubuntu installer
**As a** homelab owner
**I want** to create a bootable Ubuntu Server 24.04 LTS USB drive from my Mac
**So that** I can perform a clean install on HOLYGRAIL

#### Acceptance Criteria
- Ubuntu Server 24.04 LTS ISO downloaded and verified (SHA256)
- Bootable USB created using balenaEtcher or `dd`
- USB tested to boot successfully on HOLYGRAIL

### US-3: Install Ubuntu Server with headless defaults
**As a** homelab owner
**I want** to install Ubuntu Server 24.04 LTS with OpenSSH and minimal packages
**So that** HOLYGRAIL is ready for remote management without a monitor

#### Acceptance Criteria
- Ubuntu Server 24.04 LTS installed (minimal install)
- OpenSSH server enabled and accessible on LAN
- Root login disabled, `john` user created with sudo

### US-4: Configure hostname, static IP, and timezone
**As a** homelab owner
**I want** HOLYGRAIL to have a stable hostname and IP on the 192.168.10.0/24 network
**So that** all other devices can reliably reach it

#### Acceptance Criteria
- Hostname set to `holygrail`
- Static IP assigned on 192.168.10.0/24 (via netplan)
- Timezone set to local timezone
- DNS pointing to Pi-hole (192.168.10.105)
