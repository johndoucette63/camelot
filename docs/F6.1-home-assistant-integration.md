## Overview

Integrate Home Assistant into the Network Advisor so the dashboard surfaces IoT device states, Thread network health, and automation status. Enable bidirectional communication: advisor reads HA state and pushes notifications back to HA.

## User Stories

### US-1: Document Home Assistant setup
**As a** Camelot admin
**I want** the HA installation documented (which Pi, IP, integrations, Thread topology)
**So that** the advisor integration is built on accurate assumptions

#### Acceptance Criteria
- HA device IP, OS, and version documented in INFRASTRUCTURE.md
- Key integrations listed: HomeKit, Aqara, Thread, MQTT
- Thread border routers identified (HomePods, Aqara hubs)
- Known issues documented (Thread fragmentation, etc.)

### US-2: Connect advisor to Home Assistant REST API
**As a** Camelot admin
**I want** the advisor backend to read entity states from Home Assistant
**So that** IoT device status is visible in the advisor dashboard

#### Acceptance Criteria
- HA long-lived access token configured in advisor
- Backend fetches entity states via HA REST API
- Relevant entities filtered (border routers, network devices, sensors)
- Data refreshed on a configurable interval

### US-3: Surface Thread network health in dashboard
**As a** Camelot admin
**I want** to see Thread border router status and device connectivity in the advisor
**So that** I can diagnose Thread fragmentation issues without digging through HA

#### Acceptance Criteria
- Thread border routers listed with status (online/offline)
- Thread device count and connectivity shown
- Known fragmentation issues highlighted
- AI chat can answer Thread-related questions using this data

### US-4: Push advisor notifications to Home Assistant
**As a** Camelot admin
**I want** critical advisor alerts sent as HA notifications
**So that** I get alerted on my phone via the HA companion app

#### Acceptance Criteria
- Advisor sends notifications to HA via REST API or webhook
- Notification includes: title, message, severity, affected device
- Only critical alerts pushed (configurable threshold)
- Integration tested end-to-end: advisor alert → HA → phone notification
